Chatterbox

A chat app that allows the user to send, receive, and delete messages. Message logs are updated via a chat API, and displayed on the screen.

Bibliocache

A web-based app that gamifies your reading selections. Based on the geocaching game, users are led around Charlotte to specific locations that send downloadable digital books when they are within the designated range.

Menu App

An application meant to mimic a restaurants online menu. It was built out using views in a single HTML file as well as mustache to render items to the DOM.

Netflix Binge App

Movie website site built using AngularJS pulling information from the Movie Database API. Items were taken from the API with AJAX requests.
